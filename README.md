# Food ML Projects

Machine learning projects focused on food processing, quality analysis, and operational optimization.

---

## Project 1: Water Quality Prediction

### Objective
To analyze water quality data and build a predictive model for classification.

### Tools
Python, Pandas, Matplotlib, Scikit-learn

### Work Done
- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Feature importance analysis  
- Model development and evaluation  

### Results
- Achieved ~95% prediction accuracy  
- Identified key factors affecting water quality  

---

## Project 2: Food Delivery Time Prediction

### Objective
To predict food delivery time using machine learning based on distance, traffic, weather, and operational factors.

### Tools
Python, Pandas, NumPy, Scikit-learn, Matplotlib

### Models Used
- Random Forest Regressor  
- Gradient Boosting Regressor  

### Work Done
- Data preprocessing and feature encoding  
- Exploratory Data Analysis (EDA)  
- Model training and comparison  
- Performance evaluation using MAE and MSE  

### Key Insights
- Delivery time increases with distance and traffic  
- Weather conditions affect delivery performance  
- Gradient Boosting showed improved prediction accuracy
